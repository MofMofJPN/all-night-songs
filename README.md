## このサイトについて

Pococha 配信者 **ゑうたん** のレパートリーから、今夜の気分にぴったりの1曲を探せるサイトです。

3問の気分アンケートに答えるだけで、156曲の中からベストマッチな曲を提案します。  
配信中にスマホでサッと使えるよう、スマホファースト・軽量で設計しています。

**→ [サイトを開く](https://mofmofjpn.github.io/all-night-songs/)**

---

## 機能一覧

| 機能 | 説明 |
|---|---|
| 🎯 **気分診断** | 気分・テンポ・シーンの3問でベスト5曲を提案 |
| 🎵 **全曲リスト** | 検索・五十音・タグフィルターで156曲を探す |
| ❤️ **お気に入り** | 気になる曲をブックマーク保存 |
| 🌙 **聴いた記録** | 歌ってもらえた曲を回数記録 |
| 🎲 **シャッフル** | ランダムに1曲を表示 |
| 📋 **リクエスト文コピー** | チャットに貼れる文章をワンタップでコピー |

---

## 使い方

1. **[ 今夜の1曲を探す ]** をタップ
2. 3問の質問に答える（気分 → テンポ → シーン）
3. 提案された曲の 📋 ボタンを押してリクエスト文をコピー
4. 配信のチャットに貼り付けてリクエスト！

## ローカルで動かす

`fetch()` で `data/songs.json` を読み込むため、ローカルでは HTTP サーバーが必要です。

```bash
# Python（標準搭載）
python3 -m http.server 8080

# Node.js がある場合
npx serve .
```

ブラウザで `http://localhost:8080` を開きます。

## 曲データの更新

`data/songs.json` に追記するだけです（ID は連番で採番）。

```json
{
  "id": "157",
  "title": "曲名",
  "artist": "アーティスト名",
  "kanaSection": "あ",
  "tags": {
    "mood": ["切ない"],
    "tempo": "ミドル",
    "scene": ["泣きたい夜"]
  }
}
```

タグの選択肢：
- **mood（気分）**: `切ない` / `エモい` / `元気・前向き` / `盛り上がる` / `クール・かっこいい` / `優しい・癒し`
- **tempo（テンポ）**: `スロー` / `ミドル` / `アップ`
- **scene（シーン）**: `しっとり聴きたい` / `一緒に盛り上がりたい` / `泣きたい夜` / `朝・お出かけ気分`
