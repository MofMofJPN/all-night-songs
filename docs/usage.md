## はじめに

このサイトは、ゑうたんが歌える曲のなかから、リスナーさんが「今聴きたい1曲」を見つけてリクエストできるように作りました。

スマホで配信を聴きながら、サッと使えます。
ぜんぶ無料、登録もいりません。

---

## 🎤 まずはこの3ステップから

### ① 曲を探す
ふたつの方法があります。

**🔍 聴きたい曲が決まってるとき**
`[ 全曲リスト ]` をタップ → 検索バーに曲名やアーティスト名を入れる

**✨ 気分で選びたいとき**
`[ 今夜の1曲を探す ]` をタップ → 3つの質問に答えるだけ

質問はこの3つ:
- 今の気分は？（切ない / エモい / 元気 / 盛り上がる / かっこいい / 優しい）
- テンポは？（スロー / ミドル / アップ）
- どんなシーンで聴きたい？（しっとり / 盛り上がり / 泣きたい夜 / 朝・お出かけ）

### ② リクエスト文をコピー
気になった曲が見つかったら、曲カードの **📋アイコン** をタップ。

```
○○ お願いします🎤
#ゑうえうに #ALLNIGHTSONGS
```

がクリップボードにコピーされるので、そのまま配信のコメント欄に貼ってね。

### ③ お気に入りに保存
気に入った曲は **❤️アイコン** をタップ。

あとから `[ お気に入り ]` ページで一覧できます。

---

## 🌙 シーン別 おすすめの使い方

### 「今すぐリクエストしたい！」配信中の使い方
1. トップ画面で `[ 今夜の1曲を探す ]`
2. 3問サクッと答える
3. 結果画面で気になる曲の 📋 をタップ
4. コメント欄にペースト → 送信

3問だけだから、配信を聴きながらでも30秒で完了。

### 「あの曲リクエストしたい」曲名がわかってるとき
1. `[ 全曲リスト ]` をタップ
2. 検索バーに曲名 or アーティスト名を入力
3. 出てきた曲の 📋 をタップ

検索は**部分一致**なので、うろ覚えでも大丈夫。
五十音インデックス（あ・か・さ…）でも絞れます。

### ゆっくり曲を眺めたい・お気に入りを作りたい
配信前のおだやかな時間に。
1. `[ 全曲リスト ]` でスクロール
2. 気になった曲は ❤️ でお気に入りに
3. 配信が始まったら `[ お気に入り ]` から選んでリクエスト

### 「次は何聴こう」迷ったとき
`[ シャッフル ]` でランダムに1曲。
新しい曲との出会いに。

### 歌ってもらった曲を記録したい
ゑうたんに歌ってもらえたら、結果画面の **✅アイコン** で記録。

`[ 聴いた記録 ]`（🌙歌ってもらった曲リスト）で、いつ何回聴いたかを振り返れます。
何回でも記録できるので、お気に入りを繰り返しお願いしたときも安心。

## ✨ 知っておくと便利なこと

### 同じ気分で診断しても、毎回ちがう曲が出ます
「もう一度診断」を押すと、**直前に出た5曲は次回除外**されます。
気分が変わらなくても、新しい曲に出会えるよ。

### 曲カードのバッジに注目
- `Key: -1` などの**キー情報**（音域調整のメモ）
- `1番のみ` `練習中` などの**状態バッジ**

リクエスト前にチェックしておくと、お互いラクです。

### 試聴したいとき
曲カードの 🎧 🍎 ▶️ ボタンから、それぞれのサービスの検索結果に飛べます。
- 🎧 Spotify
- 🍎 Apple Music
- ▶️ YouTube Music

「どんな曲か知らないけど、雰囲気で選んだから一度聴いてみたい」ときに。

### Xでシェアできます
結果画面の 🐦 ボタンで、見つかった曲をXに投稿できるよ。
「今夜はこの曲お願いしたい！」を共有して、ゑうえうにを盛り上げよう。

## 💾 データについて大切なこと

お気に入り・聴いた記録・メモは、**お使いのブラウザに保存**されます。
サーバーには送られません（つまり、運営側でも見えません）。

### こんなことに気をつけてね
- 📱 スマホとPCなど、**端末をまたいで見ることはできません**
- 🗑 ブラウザの履歴・キャッシュを消すと、**保存データも消えます**
- 🕵️ シークレットモード／プライベートブラウジングでは**保存されません**
- ⏳ 容量や設定により、自動で消えることがあります

### 大切なデータはエクスポートを
`[ About ]` 画面の「データを書き出す」ボタンで、お気に入りや聴いた記録を**JSON形式でダウンロード**できます。

機種変更やブラウザ移行のときは、こちらで保存してから「データを読み込む」で復元してね。
